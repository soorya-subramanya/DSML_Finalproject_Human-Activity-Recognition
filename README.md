 This project focuses on classifying human physical activities such as walking, sitting, standing, and more using sensor-based data. The dataset consists of numerical features collected from wearable devices, which are used to train machine learning models for activity prediction.

The workflow includes data preprocessing, exploratory data analysis (EDA), feature selection, and model building. Multiple classification algorithms such as Logistic Regression, Decision Tree, K-Nearest Neighbors (KNN), Support Vector Machine (SVM), and Random Forest were implemented and compared.

Among all models, Random Forest achieved the best performance with an accuracy of around 93%, demonstrating its effectiveness in handling complex patterns in sensor data.

This project highlights the application of machine learning in healthcare and fitness analytics, with potential use cases in smart devices, activity tracking, and health monitoring systems.
